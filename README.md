# BT-5240-Computational-Systems-Biology
This repository contains all the material related to the course BT5240 Computational Systems Biology taught by Prof. Karthik Raman during the January-May 2026 semester at IIT Madras. The course consisted of four assignments and a course project.
<br>
</br>

| Assignment | Description |
| --- | --- |
| 1 | Introduction to Mathematical Modelling | 
| 2 | Network Biology using NetworkX | 
| 3 | Constraint-Based Modelling using COBRAPy | 
| 4 | Dynamic Modelling |


For the model reviewed for Assignment 1, see [here](https://doi.org/10.1111/j.1432-1033.1968.tb00175.x).
